Gunning Over It

About this game:

This game was developed as part of the Friday Night Game Jam in which you have 3 hours development time. I probably enjoyed developing this game more than any for the Friday Night Game Jam maybe besides Fish Fetcher or The Maze. This is one of my first games for the Friday Night Game Jam too and is a recreation of a previous Friday Night Game Jam entry created by Danial Jumagaliyev named Shooting Over It. Mine version is arguably worse in terms of programming but in my personal opinion has more content as i updated it post jam. The assets are from a Kenny asset pack known as Monochrome 1 bit. Furthermore, Dan (Danial) was the at the time manager of my main Indie game development team known as Team Melon and you can find his website here: https://www.danqzq.games/

You can play the game at my itch.io page: bigmemerman1.itch.io
