BMOD

About this game:

BMOD was one the first games after Fish Fetcher that I made for the Friday Nigth Game Jam. In my opinion, this game is rather impressive due to the 3 hour development limit i had to work under. Despite the fact a lot of the assets belong to sketchfab, i had implemented a detailed UI with a spawn menu that included multiple weapons, props, npcs and ragdolls. This game was updated post game jam to clean up the UI and add a flat grass map. I think this game specifically is only really impressive due to the sheer amount of content i managed to cram into it for the three hour time limit. And yes this was heavily "Inspired" by Garry's Mod. (I'm an uncreative fuck)
