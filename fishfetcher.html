Fish Fetcher

About this game:

Fish Fetcher was one my first games I made after a long hiatus from game development and programming in general. I made it for the Friday Night Game Jam which is a game jam/competition in which you must make a game within a three hour development time frame over the course of a weekend. It's a simple, semi-idle game in which you must collect fish and exchange said fish for upgrades and items to further your fishing abilties. I used a limited colour palette as I wanted to focus more on programming than art due to the time constraints. Everything in the game is rather basic but a few primary talking points include: The spawning system, ending and ui. The spawning system was quite inefficient and actually quite predictable as it was not random. It was an empty object that held a script which spawned the fish but rather than calculating where to spawn the fish randomly. The object had an animator attatched to it which rapidly moved the object around. This was due to my lack of knowledge at the time and the time constraints for development. Now, the ending occurred when a certain number of fish were met. It faded to white, displayed a small amount of text congratulating the player and thanking them for playing the game. The UI too, it doesn't scale correctly to the screen size so if you were to play the game on a different resolution to the preset webgl resolution, the ui would be bugged. Thats all, it was fun developing this game as it was a return to form and a great learning experience.
